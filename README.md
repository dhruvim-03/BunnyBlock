# 🌟 About The Game

**BunnyBlock** is a lightweight browser-based endless runner game where players control a bunny and avoid incoming obstacles for as long as possible.

The project demonstrates how engaging gameplay and smooth animations can be built entirely using fundamental web technologies without external frameworks or game engines.

Simple to understand, easy to play, and surprisingly addictive.

# 🚀 Running Locally

### Clone Repository

```bash
git clone https://github.com/dhruvim-03/BunnyBlock.git
cd BunnyBlock
```

### Run

Simply open:

```bash
index.html
```

or

```bash
npx serve .
```

Then visit:

```bash
http://localhost:3000
```
